# Aplicación de gestión de eventos científicos

Aplicación diseñada para facilitar la organización, registro y seguimiento de eventos científicos como congresos, seminarios y conferencias académicas.

## Funcionalidades previstas

- **Registro de eventos**: Crear y gestionar información de eventos científicos (título, fecha, ubicación, descripción)
- **Gestión de participantes**: Inscripción y administración de asistentes, ponentes y organizadores
- **Control de ponencias**: Envío, revisión y programación de comunicaciones científicas
